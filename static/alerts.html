How to Create a Discord Webhook

  1. Open Discord and right-click your server name at the top of the channel list. Select Server Settings.
  2. In the left sidebar under Apps, click Integrations.
  3. Click Webhooks, then click New Webhook.
  4. Give it a name (e.g. "Validator Alerts"), and use the Channel dropdown to pick which channel receives alerts.
  5. Click Copy Webhook URL — paste it into the form above.

Webhook URLs look like: https://discord.com/api/webhooks/123.../abc...
Note: Do NOT use the gear icon at the bottom-left — that opens User Settings, not Server Settings.
